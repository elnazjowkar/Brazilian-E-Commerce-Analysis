## Project Overview

This project analyzes Brazilian e-commerce data using Python and Power BI.

### Objectives
- Analyze customer behavior
- Identify sales trends
- Evaluate payment methods
- Measure seller performance
- Generate business insights through data visualization

### Tools Used
- Python
- Pandas
- Power BI
- GitHub

### Files
- analysis.py
- customers_analysis.py
- products_analysis.py
- payments_analysis.py
- seller_analysis.py
- monthly_trend_analysis.py
